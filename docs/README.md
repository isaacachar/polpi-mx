# Polpi MX - Static Site

This is the GitHub Pages deployment of Polpi MX, a real estate intelligence platform for Ciudad de MÃ©xico (CDMX).

## About

Polpi MX provides market analysis and property listings for CDMX neighborhoods. This static version includes:

- 83 real property listings from Lamudi
- Interactive map with Mapbox GL JS
- Property filters and search
- Deal scoring and analysis
- Investment calculator
- Neighborhood comparisons

## Data

The data is embedded in JSON files for static hosting:
- `js/data-listings.json` - All property listings
- `js/data-stats.json` - Market statistics

Data was scraped on February 3, 2026 and represents a snapshot of the CDMX real estate market.

## Technology

- Pure HTML/CSS/JavaScript (no backend)
- Mapbox GL JS for interactive maps
- Chart.js for visualizations
- Responsive design
